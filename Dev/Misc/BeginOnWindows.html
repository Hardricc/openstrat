Contributing to opensource for absolute beginners on Windows

This is not completely accurate yet.

  1. Open a github account. You will just need to create a user name, a password and give an email address
  2. Go to https://github.com/Richtype/openstrat and click on the "Fork" button toward the top right of the page
  3. Go to the home folder (or where ever else you want to create your clone of the project) on your desktop in Windows and open a bash terminal in that folder. And then from that terminal run the following commands:
  4. git clone https://github.com/Richtype/openstrat.git
  5. git branch bSarah1
  6. Open the file GitPractice.txt in the editor of your choice. This GitPractice.txt file is created purely for the purpose of practicing pull requests and testing git / github workflows, so you can edit it without worrying about damaging anything important.
  7. Just add / insert a message where ever you fancy in the file.
  8. Save this file. Then from the root folder of the project
  9. git add Misc/GitPractice.txt
  10. git commit -m "Sarah's first commit"
  11. git push origin master
